Faculty Development Program
on

"Research Trends,Techniques and Opportunities In Data and Text Analytics With Machine Learning."

About event

The machine learning play significant role in modern engineering and information processing systems. The objective of FDP is to present recent trends and applications of machine learning. This FDP would provide participants with the guidelines to explore the area of machine learning and its applications. The program is intented to provide opportunity to participants for upgrading their knowledge in the area of Data analytics and machine learning.
This programme will introduce to participants the various skills that go into machine learning,provide practical walk-through of relevant languages,tools and lay down a study plan for moving forward in this field.

Schedule

First day of workshop

Session 1

  • Introduction to data science with R and Python,Data exploration.


Session 2

  • Data Manipulation,Data visualization


Session 3 (Hands On)

  • Installation of R studio, implementing simple mathematical operations and logic using R operators, modifying and extracting the results from the dataset using user-defined functions in R.


  • Creating data visualization to understand the customer churn ratio using charts using ggplot.

Second day of workshop

Session 1

  • Machine learning concepts.


Session 2

  • Logistic Regression,Poisson Regression,ROC curve.


Session 3 (Hands On)

  • Implementing Linear and Logistics Regression in R by building model with 'tenure' as dependent variable and multiple independent variables.


  • Implementing predictive analytics by describing the data and explaining the relationship between one dependent binary variable and one or more binary variables.

Third day of workshop

Session 1

  • Decision trees classifier, confusion matrix,KNN algorithm.


Session 2

  • Support vector machine, Random Forest, XGBOOST.


Session 3 (Hands On)

  • Implementing random forest for booth regression and classification problems.


  • Build a tree, prune it by using 'churn' as the dependent variable.


  • Build a random forest with the right number of trees, using ROCR for performance metrics.



Fourth day of workshop

Session 1

  • Unsupervised learning: Canopy Clustering, k-means clustering algorithm.


Session 2

  • Principal component analysis, types of PCA.


Session 3 (Hands On)

  • Deploying unsupervised lerning with R to achieve clustering and dimensionally reducton.


  • K-means clustering for visualizing and interpreting results for the customer churn data.



Fifth day of workshop

Session 1

  • Association rule mining and market basket analysis, Apriori algorithm,Recommendation Engine in R.


Session 2

  • Time series analysis, Arima model, sentiment analysis in R(Twitter sentiment analysis).


Session 3 (Hands On)

  • Deploying association analysis as a rule-based machine learning method,identifying strong rules discovered in databases with measures based on intresting discoveries.


  • Analyzing time series data, sequence of measurements that follow a non-random order to identify the nature of phenomenon and to forecast the future values in the series.
